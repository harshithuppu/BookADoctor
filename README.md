🩺 Book A Doctor

📖 Project Overview

Book A Doctor is a web-based healthcare appointment booking system that allows patients to find doctors and schedule appointments online. The application simplifies the appointment booking process and provides separate functionalities for Users, Doctors, and Administrators.

---

🎯 Objectives

- Simplify doctor appointment booking.
- Provide secure user authentication.
- Enable doctors to manage appointments efficiently.
- Allow administrators to manage users and doctors.
- Maintain appointment records digitally.

---

✨ Features

👤 User Features

- User Registration
- User Login
- Browse Doctors
- Book Appointments
- View Appointment History
- Manage Profile

🩺 Doctor Features

- Apply as Doctor
- Manage Doctor Profile
- View Appointments
- Approve or Reject Appointments

🔐 Admin Features

- View All Users
- View Doctor Applications
- Approve or Reject Doctors
- Manage Platform Data

🔄 System Workflow

1. User registers and logs in.
2. User browses available doctors.
3. User books an appointment.
4. Doctor receives appointment request.
5. Doctor approves or rejects appointment.
6. Admin manages doctors and users.
7. Appointment history is stored in the database.
